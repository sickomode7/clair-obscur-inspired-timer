# Expedition 33 — Temporal Rite

A beautifully gamified, Clair Obscur: Expedition 33-inspired Pomodoro timer and quest board application built in a single HTML file. 

## Features

- **Immersive Aesthetic**: Stunning, rich visuals including painted skies, a glowing UI, floating particles, and intricate "sepia/parchment" themes.
- **Gamified Timer**: Uses the Pomodoro technique to divide work into time blocks, separated by short breaks. 
  - Complete focus sessions to gain XP and restore HP.
  - Advance through levels (Wanderer → Chronicler → Cartographer → Sentinel, etc.)
  - Track session "Gems" for your progress throughout the day.
- **Quest Board (Task Manager)**: Enlist new tasks, clear them for direct XP, and manage your objectives.
- **Responsive Design**: Adapts beautifully to both desktop and mobile screens.

## Getting Started

Since the entire application is bundled within a single file (`index.html`), setup is extremely simple:

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Alternatively, you can deploy it instantly on static hosting platforms like Vercel, Netlify, or GitHub Pages.

## How to Use

1. **Focus Sessions**: Start the timer (default 25 minutes) to begin a focus session. Earning XP grants you Level-Ups over time!
2. **Quest Board**: Click "Quests" (or look at the side panel) to add tasks. Checking off tasks rewards immediate XP.
3. **Pacing**: Make sure not to work too long without breaks, or your HP might deplete!

## Technologies Used

- Vanilla HTML5
- Vanilla CSS3 (Custom properties, grid, flexbox, keyframe animations)
- Vanilla JavaScript (Canvas API for background particles, dynamic DOM manipulation)
